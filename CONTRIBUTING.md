# Contributing

Thanks for helping improve SuperHero Agent.

## Project areas

- `packs/source/SuperHeroAgent_BP` — behavior pack and ownership script
- `packs/source/SuperHeroAgent_RP` — visuals, geometry, language files
- `extensions/RaiseSuperHeroAgent` — MakeCode extension

## Development workflow

1. Edit the source files
2. Rebuild the `.mcpack` and `.mcaddon` release artifacts
3. Test inside Minecraft Education
4. Update `CHANGELOG.md` when behavior changes

## Design principles

- Prefer gameplay-native interactions over command-based features
- Keep Member users in mind first
- Preserve classroom safety and avoid destructive mechanics
