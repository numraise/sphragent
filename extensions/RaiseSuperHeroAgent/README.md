# RaiseSuperHeroAgent

Member-safe MakeCode helper blocks for classroom activities built around the SuperHero Agent add-on in Minecraft Education.

## SuperHero gameplay

- Each player can keep only one SuperHero Agent.
- A newly spawned hero is automatically claimed by the nearest player.
- The hero shows the owner's player name above its head.
- The hero protects its owner and attacks nearby hostile mobs.
- The hero can be hurt and can die.
- Feed with `bread`, `cooked beef`, or `golden apple` to heal.
- Upgrade with `iron ingot`, `gold ingot`, and `diamond`.

## Why this extension is Member-safe

This extension intentionally avoids slash commands.  
It only provides lesson-flow blocks such as instructions, mission text, countdowns, checkpoints, and victory messages.

## Import into MakeCode

1. Publish this folder as a GitHub repository.
2. In MakeCode for Minecraft, open **Extensions**.
3. Choose **GitHub URL**.
4. Paste the repository URL.
